# Progressbar

`Progressbar` is a console-based progress bar for Java with minimal runtime overhead.

#### Installation

Depending on your build tool, add the following setting.

=== "Maven"

    ``` xml
    <dependency>
        <groupId>me.tongfei</groupId>
        <artifactId>progressbar</artifactId>
        <version>$VERSION</version>
    </dependency>
    ```

=== "Gradle"

    ``` groovy
    compile 'me.tongfei:progressbar:$VERSION'
    ```

The newest `$VERSION` is [![Maven Central](https://img.shields.io/maven-central/v/me.tongfei/progressbar.svg?style=flat-square)](https://maven-badges.herokuapp.com/maven-central/me.tongfei/progressbar).

#### Getting started

``` java
import me.tongfei.progressbar.*;
```

And then refer to [Declarative usage](declarative-usage.md) or [Imperative usage](imperative-usage.md), two flavors of using a progress bar.
